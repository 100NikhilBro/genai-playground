# Steps to setup python for Gen-AI

1. Install Python 
3. Create a folder  - GenAi  or whatever name u want 
3. Create a Virtual Environment inside your root folder 
4. To Create Virtual Environment - Run - ```python -m venv venv ```
5. Now Activate the Virtual Environment - Run - ``` .\venv\Scripts\Activate.ps1```
6. Do Whatever Experiment u want to do 

- 1: To run a .py file - ```python fileName```
- 2: To Decativate the venv - ``` deactivate``` and for activate again run ``` \venv\Scripts\Activate.ps```
